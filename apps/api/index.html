Stupid Search App API

Run Stupid Search Apps from YOUR WEBSITE

How to do it:
Use JQuery to make a GET request to our app source file.
Example code that runs a hello world app:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>

<script>
$(document).ready(function(){

$.get("https://stupidsearch.github.io/Imnotdumb/apps/test.js", function(data, status){
eval(data);
});

});
</script> var id = "YOURAPIKEYHERE"; See it running live
